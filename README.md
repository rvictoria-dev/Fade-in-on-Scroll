# Fade-in-on-Scroll

### ✨ About

***Fade In On Scroll*** is a front-end project that uses the IntersectionObserver API to trigger smooth fade-in animations as elements enter the viewport. It showcases the integration of JavaScript and CSS to create responsive and performant user interface interactions.

---

### 💡 What i learned

- How **JavaScript and CSS** work together, with JavaScript controlling when animations happen and CSS handling the visual effects.
- What the **IntersectionObserver API** is and how it can detect when elements enter or leave the viewport.
- The difference between ``entries`` and ``entry``, and how they provide information about observed elements.
- How ``entry.target`` refers to the specific HTML element being observed.
- What an observer is and how ``unobserve()`` can stop tracking elements after their animation has been triggered.
- How the ``threshold`` property determines how much of an element must be visible before the observer activates.
- How to create smooth and efficient scroll animations without using continuous scroll event listeners.
